  • Data Types

    In Javascript, there are multiple ways to specify data. Strings, integers/numbers and boolean values are the simplest forms.

    Strings are a collection of letters, numbers and punctuation. You can write anything in a string as long as you put quotation marks around it.

    Integers/Numbers are actual numbers. They do not need quotation marks.

    Boolean values are true and false. They are often used for comparison purposes.

    							
								"Hello World!"
								100
								true
								false

  • Comments

    When writing an form of code, it's important to document what we are doing with notes and comments. Comments in Javascript are ignored when Javascript is run and are simply used for developer usage.

    There are two methods to write comments in javascript. To write a single line comment, use two forward-slashes - //.

    Multi-line comments are represented by an opening /* and closing */. Any text or code between those lines will be ignored by Javascript.

    							
							// This is single-line comment

							/*
							This is a multiline comment.
							It is used to write paragraphs
							of text within your code.
							*/

  • Brackets

    Javascript uses different types of brackets within it's syntax. Each type of brackets displays a different class of data and when reading code, the brackets can help to understand the programming path.

    Square brackets are used to notate an object or the opening of a block statement.

    Curly Braces are used to notate an Object.

    Rounded brackets are used to accept arguments and create placeholders for data.

    							
							[] // Array
							{} // Object or block statement
							() // Parentheses

  • Variables

    Within Javascript and many other programming languages, data can be stored in what are called variables.

    Variables are used to store information that may be used often, or store the result of a computational equation.

    To declare a new variable, you must declare the construct with the term var and than a name.

    In order to assign data to the variable, use a single equals sign followed by a string, integer, boolean, object or array.

    To call the variable, simply call it by the name you assigned to it.

    							
								var myFirstVariable = "Hello World!";
								myFirstVariable; // Returns "Hello World!"

								var mySecondVariable = 99;
								mySecondVariable; //Returns 99

  • Console

    Since Javascript works mainly behind the scenes, it's important for us to see what is happening in our browser. By using console.log(); we can have messages and data show up in our browsers console.

    Within the parentheses of the statement, you can pass in a variable, string, number or boolean and have that value returned within the browsers console.

    							
							console.log("Hello, I'm a string!");
							// Outputs to the console
							"Hello, I'm a string!"

							console.log(061314);
							// Outputs to the console
							061314

  • Computational Operators

    Specific characters can be used to perform mathematical equations, combine data and increment/decrement numbers

    The plus character is used for adding two values.

    The plus character can also be used to attach values to each other. This is called concatenation

    The minus character is used for minusing one value from another

    The asterisk character is used for multiplying values.

    The forward slash character is used for division.

    Two plus signs are used to represent incrementation

    Two minus signs are used to represent decrementation

    							
								5 + 10
								// Outputs 15

								"Hello" + "World!"
								// Outputs "Hello World!"

								2 - 7
								// Outputs -5

								7 * 5
								// Outputs 35

								15 / 3
								// Outputs 5

  • Concatenation

    Data types can be combined to create a new value through concatenation. The operator used for concatenation is a plus sign.

    To combine two or more strings into one, simply use the + operator in between. The output will be one string.

    Adding a number and a boolean value to a string can be acheived through concatenation as well, however, note that the resulting data will be a string.

    When concatenating data, make note that no spaces will be created. For example, "hello" + "world" will result in "helloworld". Add a space where needed to each string, or create an empty string with a space.

    							
								"Hello" + "World"
								// Outputs "HelloWorld"

								"I'm" + 29
								// Outputs "I'm29"

								"Hi, my name is " + "Drew"
								// Outputs "Hi, my name is Drew"

								"I live in" + " " + "Toronto"
								// Outputs "I live in Toronto"

  • Comparison Operators

    Within Javascript, certain characters and a combination of characters are used to compare data. These return boolean values which can be used to control the flow of the program

    The == characters check for equality between values. Notice the use of two equal signs to differentiate between the use of the variable assignment operator.

    != is used to check inequality between values

    In order to check equality between value and data type, use ===. For inequality, use !==.

    To see if a value is greater than another, use the > character

    To see if a value is less than another, use the < character

    To check if a value is greater than or equal to another, use the >= operator

    To check if a value is less than or equal to another, use the <= operator

    							
								"hello" == "world";
								// Outputs false

								12 === 12;
								// Outputs true

								"Hello" !== 5;
								//Outputs true

								5 > 2;
								// Outputs true

								10 < 2;
								// Outputs false

								10 >= 10;
								// Outputs true

								8 <= 12;
								// Outputs true

  • Logical Operators

    Within Javascript, certain characters and a combination of characters are used to compare data. These return boolean values which can be used to control the flow of the program

    The == characters check for equality between values. Notice the use of two equal signs to differentiate between the use of the variable assignment operator.

    !== is used to check inequality between values

    In order to check equality between value and data type, use ===.

    To see if a value is greater than another, use the > character

    To see if a value is less than another, use the < character

    To check if a value is greater than or equal to another, use the >= operator

    To check if a value is less than or equal to another, use the <= operator

    							
								(10 > 4) && (2 < 5)
								// true

								myTrueVariable !== true || 25 > 10
								// true

  • Functions

    Functions are used to create blocks of code that you can call within your code. To create a function, use the function keyword and give it a name. Calling the function involves using the name you created.

    Within the parentheses, you have the ability to create placeholders for arguments. These placeholders are variables that are called within the function. When calling the function, you can pass data into the parentheses to be evaluated. If no arguments are to be passed, simply leave the brackets empty.

    Following the parentheses are an opening and closing pair of curly braces that indicate the code to be evaluated when the function is called. Everything inside of these curly braces is called a block statement.

    The alternate syntax for creating a function is to declare a variable with a name and set the value of it to a function.

    							

								function sayHello() {
								  console.log("Hi!");
								}

								var addMe = function(a,b){
								  return a + b;
								}

								sayHello();
								// Outputs "Hi!"

								addMe(2,4);
								// Outputs 8

  • Array

    An array is a collection of values. To declare an array, create a variable with a name and set the value to opening and closing square brackets. Inside the brackets, all data types can be accepted, but must be seperated by a comma.

    To access data within an array, call the variable name you assigned to array. You will be returned the entire array with the square brackets.

    To access a specific value inside of the array, call the variable name you assigned but append opening and closing square brackets with a number value inside, that corresponds to the index of the data inside. Note that arrays use Index-based ordering, so all values start at 0 and move incrementally.

    							
								var myFirstArray = ["string", 99,  true];

								console.log(myFirstArray);
								// Outputs ["string", 99,  true]

								console.log(myFirstArray[1]);
								// Outputs 99

  • Object

    Similar to an array, is an object. An object contains data, much like an array however, they are represented as properties and values. These are called key/value pairs. Each pair is seperated by a comma.

    To declare an object, create a variable with a name and set it's value to a set of opening and closing curly braces. Inside, declare a property and use a colon to define it's value. The value can be any data type, a variable you've created previously, an array, or a function.

    To access the data inside of an object, you can simply call the object name and you will get back the object as a whole. To access specific properties inside, call the object by name, but append the property name with a period. ie object.propertyName.

    To access data that is the stored in an array as the value of a property, use the square bracket notation from the array syntax to have the value returned. object.propertyName[i].

    A property can also hold another object and is accessed through the same syntax as before, simply reference the heirarchy of the object by name. These objects are referred to as Nested Objects.

    							
							var myFirstObject = {
							  itemOne: "String",
							  itemTwo: 1027,
							  itemThree:true,
							  itemFour: ["Hello", "Goodbye"],
							  itemFive: {
							    subItemOne: "I'm a nested object!",
							    subItemTwo: "So am I!"
							  }
							};

							console.log(myFirstObject);
							// Outputs Object {itemOne: "String", itemTwo: 1027, itemThree: true, itemFour: ["Hello", "Goodbye"]}

							console.log(myFirstObject.itemOne);
							// Outputs "String"

							console.log(myFirstObject.itemFour[2]);
							// Outputs "Goodbye"

							console.log(myFirstObject.itemFive.subItemOne);
							// Outputs "I'm a nested object!"

  • Method

    One of the powers of an object is the ability to set the value of a property to a function. This resulting key/value pair is called a method.

    Just like the alternate syntax for creating a function when a variable is created and assigned a function, a method is referenced by a property name that has a value of a function.

    When creating a method, the parentheses are required after the function keyword and can be empty or have placeholders for arguments to be passed.

    To call a method, the object syntax is still used, however, you must use the property name with a pair of parentheses, much like calling a function.

    							
							var myObject = {
							  propertyOne: "Hello Drew",
							  propertyTwo: "Goodbye Drew",
							  propertyThree: function() {
							    console.log("I'm a method!");
							  },
							  propertyFour: function(name) {
							    console.log("Nice to meet you " + name);
							  }
							};

							console.log(myObject.propertyThree());
							// Outputs "I'm a method!"

							console.log(myObject.propertyFour("Drew"));
							// Outputs "Nice to meet you Drew"

  • Conditions

    Conditions can be used to control program flow. Using an if/else statement, you can affect the outcome of the program.

    A standard if/else statement uses an if statement followed by parentheses. Inside of the parentheses, you enter a comparison equation. Following the parentheses are opening and closing curly braces. Inside of the curly braces is a block statement containing the code to be executed.

    Following the closing curly brace you provide an else statement that includes opening and closing curly braces with a block statement inside if the initial if condition was not met.

    To provide additional conditions and executions, insert else if () statements between the initial if and the closing else statement.

    							
							if (5 > 10) {
							  console.log("hello");
							} else if (3 < 10) {
							  console.log("goodbye");
							} else {
							  console.log("see ya");
							}
