  • Data Types

    In Javascript, there are multiple ways to specify data. Strings, integers/numbers and boolean values are the simplest forms.

    Strings are a collection of letters, numbers and punctuation. You can write anything in a string as long as you put quotation marks around it.

    Integers/Numbers are actual numbers. They do not need quotation marks.

    Boolean values are true and false. They are often used for comparison purposes.

    "Hello World!"
    100
    true
    false

  • Variables

    Within Javascript and many other programming languages, data can be stored in what are called variables.

    Variables are used to store information that may be used often, or store the result of a computational equation.

    To declare a new variable, you must declare the construct with the term var and than a name.

    In order to assign data to the variable, use a single equals sign followed by a string, integer, boolean, object or array.

    To call the variable, simply call it by the name you assigned to it.

    var myFirstVariable = "Hello World!";
    myFirstVariable;

    > "Hello World!"

    var mySecondVariable = 99;
    mySecondVariable;

    > 99

  • Computational Operators

    Specific characters can be used to perform mathematical equations, combine data and increment/decrement numbers

    The plus character is used for adding two values.

    The plus character can also be used to attach values to each other. This is called concatenation

    The minus character is used for minusing one value from another

    The asterisk character is used for multiplying values.

    The forward slash character is used for division.

    Two plus signs are used to represent incrementation

    Two minus signs are used to represent decrementation

    5 + 10
    > 15

    "Hello" + "World!"
    > "Hello World!"

    2 - 7
    > -5

    7 * 5
    > 35

    15 / 3
    > 5

  • Comparison Operators

    Within Javascript, certain characters and a combination of characters are used to compare data. These return boolean values which can be used to control the flow of the program

    The == characters check for equality between values. Notice the use of two equal signs to differentiate between the use of the variable assignment operator.

    !== is used to check inequality between values

    In order to check equality between value and data type, use ===. For inequality, use !===.

    To see if a value is greater than another, use the > character

    To see if a value is less than another, use the < character

    To check if a value is greater than or equal to another, use the >= operator

    To check if a value is less than or equal to another, use the <= operator

    "hello" == "world";
    > false

    12 === 12;
    > true

    "Hello" !== 5;
    > true

    "Hello" !=== "World";
    > false

    5 > 2;
    > true

    10 < 2;
    > false

    10 >= 10;
    > true

    8 <= 12;
    > true
